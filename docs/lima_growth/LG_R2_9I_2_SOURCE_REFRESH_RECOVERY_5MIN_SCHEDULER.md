## 5. 5-MIN SCHEDULER

| Component | Status |

| Tick logic | Detects new data, runs refresh if needed |
| Auto-refresh | On tick, checks for new operational date |
| Heartbeat | Records tick even when no new data |

### Endpoints

| Method | Path | Purpose |
|--------|------|---------|
| GET | /yego-lima-growth/scheduler/status | Current scheduler state |
| POST | /yego-lima-growth/scheduler/start | Enable scheduler |
| POST | /yego-lima-growth/scheduler/stop | Disable scheduler |
| POST | /yego-lima-growth/scheduler/tick | Manual trigger (for cron) |

### External cron setup

```
*/5 * * * * curl -X POST http://localhost:8000/yego-lima-growth/scheduler/tick
```

Scheduler defaults to `enabled=false` (manual activation required).
